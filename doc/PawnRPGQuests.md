# RPG 人物对话与个人任务分析报告

## 1. 核心理念：个人化与去中心化
在 `RimChat` 的 RPG 系统中，Pawn（人物）发起的任务应与派系领袖（Leader）发起的任务有本质区别。
- **派系任务**：侧重于地缘政治、资源掠夺、领土扩张和宏观外交。
- **个人任务**：侧重于个人需求、情感纠葛、性格特质（Traits）以及微观的社交互动。

## 2. 目标分类与任务类型

### A. 目标：本派系 (Own Faction)
反映 Pawn 在其社会结构中的角色与诉求。
- **补给请求**：基于 Pawn 的职业。例如厨师请求更好的食材，医生请求稀缺药品。
- **设施升级**：请求改善其工作环境（如研究台、工作台）或个人居住环境。
- **职场调解**：请求解决与同事的矛盾，或请求与某人共同执行任务。
- **职业转型**：请求学习某种技能，需要玩家指派导师。

### B. 目标：囚犯 (Prisoners)
反映 Pawn 的性格（善良、嗜血、嫉妒等）对弱势群体的态度。
- **私下审讯**：Pawn 怀疑囚犯藏有秘密，请求单独接触（可能增加信任或恐惧）。
- **慈悲请求**：善良性格的 Pawn 请求为囚犯提供更好的食物或医疗。
- **复仇/折磨**：与囚犯有仇的 Pawn 请求“特殊照顾”该囚犯。
- **招募倡议**：Pawn 认为该囚犯是可造之才，请求加大招募力度。

### C. 目标：敌人 (Enemies)
反映个人恩怨与战术情报。
- **个人宿怨**：请求猎杀或俘虏某个特定的敌对 Pawn（杀死其亲人的凶手）。
- **寻回遗物**：请求前往敌对营地找回被抢走的传家宝或个人物品。
- **内应情报**：Pawn 的朋友想叛逃，请求玩家派人接应。
- **弱点暴露**：Pawn 声称知道敌对基地的一个具体防御漏洞，提议突袭。

### D. 目标：玩家/殖民者 (General Personal)
- **身体改造**：请求获得义体或先进假肢（基于健康需求或追求超人类主义）。
- **成瘾救助**：私下请求获得某种药物以缓解成瘾痛苦。
- **情感表达**：请求玩家帮助其追求某个殖民者，或创造相处机会。

## 3. 五维属性与任务关联 (Correlation with 5-Dimensions)

| 任务类型 | 核心驱动属性 | 成功收益 | 失败惩罚 |
| :--- | :--- | :--- | :--- |
| **职业补给** | 尊重 (Respect) | 依赖度+, 尊重+ | 尊重-, 好感- |
| **慈悲/复仇** | 好感 (Favorability) | 好感+, 信任+ | 信任-, 恐惧+ (若是拒绝复仇) |
| **个人宿怨** | 恐惧 (Fear) / 信任 | 信任++, 尊重+ | 信任--, 尊重- |
| **成瘾/身体改造** | 依赖 (Dependency) | 依赖度++, 信任+ | 依赖度-, 绝望感+ |

## 4. 区别于派系任务的特征总结

| 特征 | 派系/首领任务 | RPG 个人任务 |
| :--- | :--- | :--- |
| **发起动机** | 派系利益、生存、权力 | 性格、心情、个人历史、生理需求 |
| **影响范围** | 整个派系、世界地图 | 1-3 个具体 Pawn、局部设施 |
| **奖励类型** | 派系好感度、声望、大宗物资 | 个人五维波动、技能提升、个人装备 |
| **语言风格** | 正式、官僚、威胁、博弈 | 私人、情感化、恳求、秘密、碎碎念 |

## 5. 开发者实施建议
1. **上下文注入**：在 RPG 系统提示词中，除了派系背景，必须注入当前 Pawn 的 **性格 (Traits)**、**健康状况 (Health)**、**社交关系 (Social Relations)** 和 **职业 (Work Tags)**。
2. **任务模板升级**：当前的 `RimChat_AIQuest` 过于通用。应开发专门支持“指定目标人物”或“指定物品类型”的 QuestNode。
3. **反馈循环**：个人任务的成功应直接反馈到 Pawn 的五维属性中，而非简单的派系好感度增减。
