## 简介

RimChat为《边缘世界》带来真实的对话体验！让通讯台外交由AI控制,和人物进行沉浸式对话,你的对话将影响游戏世界.

---

## 功能

### 🤖 真实的通讯台外交对话
- 派系领袖由AI智能控制
- 根据派系特性、关系和当前局势做出动态决策
- 支持多种外交行动：贸易、结盟、宣战、求和等
- 支持AI调用的行为: 袭击、支援、发布任务等

### 💬 RPG风格的NPC对话
- 与NPC进行自然语言交流
- 根据性格、关系、记忆和环境做出动态对话
- 支持AI调用的行为: 影响心情记忆,恋爱,求婚,分手等

### 📰 动态社交圈公告板
- 实时追踪世界的重大事件
- 派系关系变化、战争动态、生态事件等
- 通过社交圈了解世界上发生的事情

### ⚡ 智能事件触发
- AI根据游戏情况主动触发事件
- 人物对话和外交对话相互影响
- 补充原版 storyteller 的故事叙述
- 更加动态和不可预测的游戏体验

## 使用前提

### 必需
- [Harmony]
- 《边缘世界》1.6
- 自备AI服务

### API配置
如需使用云端AI服务，需要在Mod设置中配置：
- API密钥
- 选择AI服务提供商 (OpenAI, DeepSeek, Google等)
- 或使用本地Player2运行模型 (推荐)

---

## 快速上手

1. 进入游戏设置 → Mod设置 → RimChat
2. 配置AI服务提供商
3. 将密钥粘贴到RimTalk设置中
4. 开始游戏，Mod会自动启用

## 推荐的免费模型

- Player2 GPT-OSS-120B
- 完全免费。没有限制，无需担心达到使用上限。
- Official website: Player2[player2.game]

## 常见问题

**Q: 为什么有时无法触发行为？**
A: 初期好感度需要达到一定水平才能解锁更多互动。

**Q: 兼容RimTalk吗？**
A: 是的,两者可以互传变量. RimTalk插件变量也可以兼容.

**Q: 兼容CE吗？**
A: 这个mod不涉及战斗机制.

**Q: 可以中途加入存档吗？**
A: 没问题。

## 开发更新（0.3.162）

- 新增：社交圈每次成功出帖后，自动将帖子摘要反写到所有活跃非玩家派系领袖记忆（`DiplomacySessionSummaries`）。
- 新增：反写摘要优先使用 `Headline + Lead`，缺失时回退 `Content`，并写入结构化 `key facts`。
- 防护：跳过 `OriginType = DiplomacySummary` 的帖子反写，避免摘要来源帖子形成自激循环。
- 稳定性：反写记录使用稳定 `ContentHash` 去重（优先 `PostId`，回退 `OriginType/OriginKey/tick`），避免重复回调导致摘要池污染。
