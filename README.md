# RimDiplomacy - AI Driven Faction Diplomacy

为 RimWorld 带来 AI 驱动的派系外交系统！

## 功能特性

- **AI 控制派系**: 让派系由 AI 控制，实现动态外交
- **智能对话系统**: 与 AI 派系领袖进行外交对话
- **动态世界新闻**: 每 2-3 天播报世界动态
- **智能事件触发**: 作为原版 Storyteller 的补充

## 构建说明

### 前置要求

- .NET Framework 4.8 SDK
- RimWorld 1.6
- Harmony Mod
